# Grupo Frucat — Web

Repositorio oficial de los sitios web de Grupo Frucat.

## Sitios

| Sitio | Dominio | Estrategia |
|---|---|---|
| **USA** | `frucat.com` | Adquisición de clientes, SEO, presencia en mercado americano |
| **México** | `mx.frucat.com` | Autoridad de marca, presencia institucional |

## Estructura

```
frucat-web/
├── usa/          # frucat.com
│   ├── index.html
│   ├── css/
│   ├── js/
│   └── _redirects
└── mx/           # mx.frucat.com
    ├── index.html
    ├── css/
    ├── js/
    └── _redirects
```

## Stack

- HTML + CSS + JS estático
- Deploy: Cloudflare Pages (2 proyectos separados)
- DNS: Cloudflare

## Cliente

**Grupo Frucat** — Comercializadora de frutas y verduras con operación en México y USA.
